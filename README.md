# 🎵 Music Player App

## Features

* Search songs
* Play / Pause / Next
* Seek bar
* Mini player
* Queue (add/remove)
* Background playback

## Tech

* React Native (Expo)
* Zustand
* Expo AV
* AsyncStorage

## Run

npm install
npx expo start

## Notes

* API fallback used for reliability
* Queue persisted locally
